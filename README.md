目前，我们正在逐步完善IntoYun SDK，争取为第三方开发者提供一个规范、简单易用、可靠、可扩展、可定制的 SDK，敬请期待。

## 快速集成 IntoYunSDK支持使用Cocoapods集成，请在Podfile中添加以下语句：

`pod "IntoYunSDK",  :git => "https://github.com/IntoYun/IntoYunSDKDemo-ios.git"`
